# Programação Orientada a Objetos
*Vamos utilizar o Git e Github para controlo de versões*
1. Criar uma conta no Github
2. Instalar [GithubDesktop](https://desktop.github.com/download/))
3. Aprender sobre [Markdown](https://www.markdownguide.org/cheat-sheet/)
4. Instalar a extensão: [Markdown All in One](https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one) no [VisualCode](https://code.visualstudio.com/sha/download?build=stable&os=win32-x64-user) 
5. Previsualizar o Markdown: Ctrl + Shift + V
